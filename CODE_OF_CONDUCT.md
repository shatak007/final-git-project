# Code of Conduct

## Our Pledge
We pledge to make participation in this project a welcoming and inclusive experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment include respectful communication and collaboration.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing the standards outlined in this Code of Conduct.

## Scope
This Code of Conduct applies within all project spaces, including issues, pull requests, and community forums.

## Enforcement
Instances of unacceptable behavior may be reported to the project maintainers for action.

## Enforcement Guidelines
Consequences for violations may include warnings, removal of privileges, or exclusion from project activities.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
